# Star Wars Canon Liste – PWA

Vollständige Star Wars Canon Leseliste (Episode I bis VI) als Progressive Web App.

## Auf iPhone installieren

1. Seite in **Safari** öffnen
2. Auf das **Teilen-Symbol** tippen (Rechteck mit Pfeil nach oben)
3. **"Zum Home-Bildschirm"** wählen
4. **"Hinzufügen"** tippen

Die App ist jetzt auf dem Home-Bildschirm und funktioniert auch offline!
Der Fortschritt wird automatisch gespeichert.

## Features

- ✅ 97 Canon-Einträge chronologisch sortiert
- ✅ Abhakfunktion mit automatischer Speicherung
- ✅ Fortschrittsbalken
- ✅ Offline-fähig (Service Worker)
- ✅ iPhone/iPad optimiert
